# FullCare Mobile - textos para Google Play

## Nome do app

FullCare Mobile

## Descricao curta

Gestao assistencial e operacional para usuarios autorizados do FullCare.

## Descricao completa

O FullCare Mobile permite que usuarios autorizados do sistema FullCare acompanhem internacoes,
consultem dados operacionais e registrem atualizacoes assistenciais diretamente pelo celular.

O aplicativo foi desenvolvido para apoiar fluxos de auditoria, acompanhamento de internados,
TUSS, prorrogacoes, evolucoes, altas, Home Care, longa permanencia e eventos adversos.

O acesso e restrito a usuarios previamente cadastrados no FullCare. As informacoes exibidas e
registradas no aplicativo seguem os perfis de acesso e as permissoes operacionais do sistema.

O FullCare Mobile trata dados pessoais e dados relacionados a saude apenas para finalidades
assistenciais, administrativas, operacionais, de auditoria e continuidade dos servicos.

## App access - instrucao para revisao

Este aplicativo exige login. Use as credenciais de teste fornecidas nesta secao do Google Play
Console para acessar o ambiente de demonstracao. A conta de teste possui permissao para navegar
pelos principais modulos sem utilizar dados reais de beneficiarios.

## Data safety - resumo para preenchimento

Dados coletados ou tratados:

- Identificadores de usuario.
- E-mail.
- Dados de autenticacao.
- Dados profissionais e perfil de acesso.
- Dados assistenciais e administrativos relacionados a internacoes.
- Dados de saude quando necessarios ao fluxo operacional.

Finalidades:

- Funcionalidade do app.
- Autenticacao e seguranca.
- Gestao assistencial e administrativa.
- Auditoria e continuidade dos servicos.
- Cumprimento de obrigacoes legais, regulatorias e contratuais.

Compartilhamento:

- Nao ha venda de dados.
- Compartilhamento apenas quando necessario para execucao dos servicos, obrigacoes legais ou
  acesso por usuarios autorizados.

Seguranca:

- Comunicacao em HTTPS.
- Login individual.
- Token de sessao.
- Restricao de acesso por perfil autorizado.

## Health apps declaration

Declarar que o aplicativo trata informacoes relacionadas a saude em contexto operacional,
assistencial e administrativo, para uso restrito por profissionais ou usuarios autorizados.

## Publico alvo

Usuarios profissionais e operacionais autorizados. O app nao e destinado a criancas.
